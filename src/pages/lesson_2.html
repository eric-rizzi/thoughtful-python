Coding Challenge: Temperature Converter

Temperature Conversion Formula

To convert from Celsius to Fahrenheit, use the formula:

F = (C × 9/5) + 32

Where F is the temperature in Fahrenheit and C is the temperature in Celsius.

Create a function called celsius_to_fahrenheit that:

  • Takes a temperature in Celsius as input
  • Converts it to Fahrenheit using the formula above
  • Returns the Fahrenheit temperature as a number
