Lesson 3: Predicting Function Output

Understanding Function Behavior

Being able to predict how a function behaves is a crucial skill for programmers. In this exercise, you'll analyze a function and predict its output for different inputs without running the code.

Predict the Output

Function: Find the Largest of Three Numbers

def find_largest(a, b, c):
    """Find the largest of three numbers"""
    if a >= b and a >= c:
        return a
    elif b >= a and b >= c:
        return b
    else:
        return c

Understanding the Logic

The find_largest function works by:

  1. Comparing a with both b and c. If a is greater than or equal to both, it returns a.
  2. If the first condition fails, it checks if b is greater than or equal to both a and c. If true, it returns b.
  3. If neither condition is true, it returns c.
